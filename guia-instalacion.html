aip-bench proxy para VS Code

Ahorra entre 40-65% de tokens al usar Claude Code en VS Code. El proxy comprime los mensajes antiguos de la conversacion antes de enviarlos a la API.
VS Code (Claude Code) → aip-proxy (localhost:8080) → api.anthropic.com
comprime mensajes aqui ↑

0 Requisitos previos

Python 3.8+, Git, VS Code y Claude Code instalado y funcionando.

Si no tienes Claude Code en VS Code: abre la terminal integrada (Ctrl+`) y ejecuta npm install -g @anthropic-ai/claude-code

1 Clonar el repositorio

Abre la terminal integrada de VS Code (Ctrl+`) y ejecuta:

cd ~
git clone https://github.com/iafiscal1212/aip-bench.git
cd aip-bench

2 Crear entorno virtual e instalar

En la misma terminal de VS Code:

python3 -m venv venv
source venv/bin/activate
pip install -e ".[proxy,anthropic]"

3 Verificar la instalacion

python -c "from aip_bench.proxy.server import ProxyServer; print('OK')"

Si ves OK, todo esta bien.

4 Configurar Claude Code para usar el proxy

echo 'export ANTHROPIC_BASE_URL=http://localhost:8080' >> ~/.bashrc
source ~/.bashrc

Esto redirige las peticiones de Claude Code al proxy local.

5 Abrir el proyecto en VS Code

Abre la carpeta del proyecto:

code ~/aip-bench
VS Code → Ctrl+Shift+P → "Python: Select Interpreter" → elige ~/aip-bench/venv/bin/python

6 Usar (todo dentro de VS Code)

Necesitas dos terminales en VS Code. Para abrir una segunda: Ctrl+Shift+`

Terminal 1 — Lanza el proxy (dejalo abierto):

cd ~/aip-bench
source venv/bin/activate
aip-proxy --port 8080 --profile balanced --verbose

Terminal 2 — Lanza Claude Code:

claude
En la Terminal 1 veras los logs en tiempo real mostrando cuantos tokens se ahorran en cada peticion.

Perfiles de compresion

Cambia --profile segun lo que necesites:

Perfil Comprime desde Ahorro Calidad Uso
conservative >8K tokens ~15% ~100% Maxima precision
balanced >5K tokens ~40% ~88% Uso diario (recomendado)
aggressive >2K tokens ~65% ~77% Sesiones muy largas

Desactivar el proxy

Temporalmente (una sola vez, en la terminal de VS Code):

ANTHROPIC_BASE_URL=https://api.anthropic.com claude

Permanentemente: edita ~/.bashrc y comenta la linea:

# export ANTHROPIC_BASE_URL=http://localhost:8080
Si tienes la variable configurada pero no lanzas el proxy, Claude Code no podra conectarse. Lanza siempre el proxy primero en la Terminal 1 o desactiva la variable.
